# Prompt Base: Pokemon Madrid V2

## Objetivo

Usar el documento fuente como contexto principal para desarrollar `Pokemon Madrid V2`, manteniendo la ambientacion en Madrid, la referencia a personajes del piso y el tono de juego personalizado.

## Documento de referencia

- Fuente principal: https://docs.google.com/document/d/1TXwBYnG3GU6KyckGEkcI4A6ZZcIZvrGQgk9g4vrDZlg/edit?usp=drivesdk

## Instruccion de trabajo

Tomar el Google Doc como base de contexto, extraer los elementos narrativos utiles y convertirlos en una version mas clara y reutilizable para futuras iteraciones del proyecto.

## Salida esperada

- Resumen limpio del contexto.
- Personajes y relaciones principales.
- Ideas reutilizables para historia, misiones o estructura de juego.
