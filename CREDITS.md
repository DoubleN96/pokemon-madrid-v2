# Credits

## Project Palladium Assets

This project currently includes selected visual assets imported from the `Project Palladium` folders inside Team Aqua's Asset Repo.

Source repository:
- https://github.com/TeamAquasHideout/Team-Aquas-Asset-Repo

Imported subsets:
- `assets/project-palladium/maps/`
- `assets/project-palladium/ui/`
- `assets/project-palladium/sprites/`

Per the repository README files:
- Team Aqua's Asset Repo presents these assets as free to use and edit by default, while asking users to respect folder-specific notes and credits.
- The `Project Palladium` folders state that Project Palladium released their assets for public free use.
- Credit is requested to the Project Palladium team, with individual credits supplied where known.

Folder-level references used for this import:
- `User Interface/Project Palladium/README.md`
- `Overworld Trainer Sprites/Project Palladium/README.md`
- `Maps/Project Palladium/README.md`

Project Palladium team credit is therefore retained here for all imported assets.
